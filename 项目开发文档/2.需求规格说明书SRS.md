## 2.2 功能需求清单

### FR-01：选品算法引擎
**优先级**：P0（核心）
**描述**：系统根据用户设定的阈值，自动从 Sorftime 数据中筛选出符合"低竞争、有利润"条件的耗材品类。
**输入**：品类关键词（如"3D打印耗材"）
**输出**：符合阈值的产品清单
**核心阈值（用户已确认）**：

| 指标 | 通用阈值 | 打印耗材特殊阈值 |
|------|---------|-----------------|
| 月总销量 | 5,000 - 150,000 | 3,000 - 200,000 |
| Listing 垄断系数 | ≤30% | ≤25% |
| 品牌垄断系数 | ≤40% | ≤35% |
| 卖家垄断系数 | ≤40% | ≤35% |
| 新品数量占比（3-6个月） | ≥5% | ≥10% |
| 新品市场份额占比 | ≥5% | ≥10% |
| 平均评论数 | ≤300条 | ≤200条 |
| 卖家/品牌数量 | ≥40个 | ≥40个 |
| 亚马逊自营占比 | ≤30% | ≤30% |
| 知识产权 | 未覆盖 | **需手动检索（系统提示）** |

**验收标准**：
- [ ] 输入"3D打印耗材"，系统能在 30 秒内返回符合阈值的产品清单
- [ ] 每个产品展示字段包含：品名、月销量、垄断系数、评论数、预估利润
- [ ] 不符合阈值的产品需显示"未通过原因"

### FR-02：1688比价监控
**优先级**：P0（核心）
**描述**：系统通过"标题模糊匹配"策略，关联 Sorftime 产品与 1688 商品，实现拿货价自动追踪。
**输入**：Sorftime 产品标题
**输出**：1688 最新拿货价、价格变动幅度、预警状态
**匹配策略**：
1. 提取核心属性（材质、规格、重量）
2. 调用 1688 `item_search` 接口
3. 按标题相似度、价格、销量排序取 Top1-3
4. 人工确认关联（提供 UI 界面）

**预警规则**：
- 当 1688 拿货价上涨 ≥5%，触发"⚠️ 成本预警"
- 当 1688 拿货价下降 ≥5%，触发"📉 降价利好"

**验收标准**：
- [ ] 系统能自动为已关联 SKU 每日刷新 1688 价格
- [ ] 价格变动超过阈值时，看板显示预警标识
- [ ] 人工确认关联界面操作流畅

### FR-03：AI市场分析日报
**优先级**：P1（重要）
**描述**：系统每天早上定时调用 GLM-5.2，基于 Sorftime 数据生成市场行情分析报告。
**输入数据**：
- 近 7 天品类销量趋势
- 竞品价格 Top 10 变动
- 当前库存成本 vs 市场售价

**AI 输出格式**（固定模板）：

```markdown
## 📊 [日期] 市场行情日报

### 🔥 今日推荐
- 产品名 + 推荐理由（2-3句）

### 📈 趋势解读
- 品类整体走向
- 值得关注的竞品动态

### ⚠️ 风险提示
- 成本变动预警
- 潜在风险品类

### 💡 行动建议
- 具体操作（调价/上架/观望）
```

**模型配置**：

| 优先级 | 模型 | 用途 |
|--------|------|------|
| 主力 | GLM-5.2 | 日常日报生成 |
| 备用1 | GLM-5.1 | 主力不可用时自动切换 |

**验收标准**：
- [ ] 每日上午 9:00 自动生成日报
- [ ] 日报包含推荐、趋势、风险、行动四个模块
- [ ] 模型切换可在配置页面一键完成

### FR-04：Web看板
**优先级**：P0（核心）
**描述**：系统的唯一前端界面，集中展示所有数据和操作入口。
**页面结构**：
1. 首页概览：今日推荐 TOP 5、预警数量、待确认关联数
2. 选品中心：符合阈值产品清单 + 筛选/排序
3. 竞品监控：你关注的竞品价格变动列表
4. AI日报：每日市场行情报告展示
5. 配置中心：阈值调整、模型切换、SKU 关联管理

**验收标准**：
- [ ] 所有页面在 PC 端流畅运行（Chrome/Edge）
- [ ] 数据刷新延迟 ≤3秒
- [ ] 看板每天首次打开自动加载最新数据

### FR-05：风险规则引擎
**优先级**：P1（重要）
**描述**：系统根据预设风险规则，自动标记"高风险"产品并给出具体提示。
**风险规则库（已确认）**：

| 品类 | 风险标签 | 提示文案 |
|------|---------|---------|
| 墨水 | 易燃液体 | ⛔ 属3类易燃液体，需MSDS+非危鉴定，空运受限 |
| 墨水 | 知识产权 | ⛔ 原装墨盒品牌备案，需提供授权书 |
| 3D耗材(树脂) | 危险品 | ⛔ 液态树脂属危险品，需避光+恒温运输 |
| 相纸 | 易损易潮 | ⚠️ 需三重包装+干燥剂，运输破损率高 |
| 打印纸 | 体积重大 | ⚠️ 体积重大，物流成本占比较高 |
| 通用 | 认证缺失 | ⚠️ 欧美市场需CE/FCC/ASTM认证 |

**验收标准**：
- [ ] 产品详情页显示所有匹配的风险标签
- [ ] 高风险产品在列表中自动标红置顶提示
- [ ] 支持用户新增/自定义风险规则

### FR-06：数据迭代与定时任务
**优先级**：P1（重要）
**描述**：系统按设定周期自动执行数据拉取和重算任务。
**任务调度**：

| 任务 | 频率 | 执行时间 |
|------|------|---------|
| Sorftime 数据拉取 | 每日 | 08:00 |
| 1688 价格刷新 | 每日 | 08:30 |
| 选品算法重算 | 每周 | 周一 09:00 |
| AI日报生成 | 每日 | 09:00 |
| 风险标签更新 | 每次数据刷新后 | 实时 |

**验收标准**：
- [ ] 定时任务准时执行，偏差 ≤5分钟
- [ ] 任务失败时发送告警通知（预留邮箱/看板提示）
- [ ] 支持手动触发重跑
