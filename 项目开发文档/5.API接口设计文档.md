# API接口设计文档

## 5.1 通用规范

| 规范项 | 说明 |
|--------|------|
| 协议 | HTTPS |
| 数据格式 | JSON (Request/Response) |
| 字符编码 | UTF-8 |
| 时间格式 | ISO 8601 (YYYY-MM-DDTHH:mm:ssZ) |
| 货币单位 | 价格字段统一为 USD（前端展示时可按需转换） |
| 分页 | page + page_size (默认20，最大100) |
| 鉴权 | 请求头 `Authorization: Bearer <token>`（单用户系统，token 为配置的访问令牌） |
| 限流 | 单 IP 60 次/分钟，超限返回 429 |
| 幂等 | 写接口支持 `Idempotency-Key` 请求头，相同 key 24h 内返回首次结果 |

### 统一响应格式

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": "2026-07-15T10:00:00Z"
}
```

### 错误码定义

| 错误码 | 含义 |
|--------|------|
| 0 | 成功 |
| 1001 | 参数校验失败 |
| 1002 | 资源不存在 |
| 2001 | API调用超时（外部） |
| 2002 | API调用失败（外部） |
| 3001 | 数据库操作失败 |
| 4001 | 无权限访问 |
| 429 | 请求过于频繁（限流） |
| 5001 | 系统内部错误 |

> FastAPI 自动生成交互式文档：`GET /docs`（Swagger）与 `GET /redoc`。

## 5.3 接口详细定义

### 5.3.1 GET /api/v1/products/
**描述**：获取产品列表，支持多条件筛选

**请求参数（Query）**：

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| category | string | 否 | 品类筛选（3d_filament/ink/photo_paper/paper） |
| match_status | string | 否 | 匹配状态（pending/confirmed/rejected） |
| risk_level | string | 否 | 风险等级（danger/warning/info） |
| min_score | float | 否 | 最低综合评分 |
| sort_by | string | 否 | 排序字段（score/price/sales） |
| sort_order | string | 否 | asc/desc（默认 desc） |
| page | int | 否 | 页码（默认 1） |
| page_size | int | 否 | 每页数量（默认 20，最大 100） |

**成功响应（200）**：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "items": [
      {
        "id": 1,
        "title": "High Quality PLA 3D Printer Filament 1.75mm Black 1kg",
        "category": "3d_filament",
        "platform": "amazon",
        "monthly_sales": 8500,
        "price": 22.99,
        "listing_monopoly": 18.5,
        "brand_monopoly": 22.3,
        "seller_monopoly": 25.1,
        "review_count": 189,
        "comprehensive_score": 87.5,
        "risk_tags": [],
        "match_status": "confirmed",
        "matched_1688_title": "厂家直销3D打印耗材PLA 1.75mm 黑色1kg",
        "estimated_profit": 5.20,
        "profit_margin": 22.6,

}
```

### 5.3.2 POST /api/v1/price/match
**描述**：手动匹配产品与 1688 商品（支持幂等，需传 `Idempotency-Key`）

**请求头**：

| 头部 | 必填 | 说明 |
|------|------|------|
| Idempotency-Key | 是 | 幂等键，24h 内重复请求返回首次结果 |

**请求体**：

```json
{
  "product_id": 1,
  "1688_item_id": "1234567890",
  "1688_item_title": "厂家直销PLA耗材",
  "search_keywords": "PLA 1.75 黑色 1kg"
}
```

**成功响应（200）**：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "product_id": 1,
    "matched_1688_id": "1234567890",
    "matched_1688_title": "厂家直销PLA耗材 1.75mm 黑色1kg",
    "price_1688": 65.00,
    "match_confidence": 92.5,
    "status": "confirmed"
  }
}
```

### 5.3.4 PUT /api/v1/config/thresholds
**描述**：更新选品阈值，修改后立即生效（无需重启）

**请求体**：

```json
{
  "monthly_sales_min": 5000,
  "monthly_sales_max": 150000,
  "listing_monopoly": 30,
  "brand_monopoly": 40,
  "seller_monopoly": 40,
  "review_count_max": 300,
  "seller_count_min": 40,
  "amazon_self_ratio_max": 30
}
```

**成功响应（200）**：

```json
{
  "code": 0,
  "message": "success",
  "data": { "updated_keys": ["monthly_sales_min", "listing_monopoly"] }
}
```

### 5.3.5 GET /api/v1/health
**描述**：健康检查（无鉴权），供 Docker healthcheck 与监控探活

**成功响应（200）**：

```json
{
  "status": "healthy",
  "db": "up",
  "redis": "up",
  "celery": "up"
}
```

> 任一依赖不可用返回 503 + `"status":"degraded"`。

## 5.4 错误码扩展

| 错误码 | 含义 | 处理建议 |
|--------|------|---------|
| 200101 | Sorftime API 调用超时 | 稍后重试（最多 3 次，指数退避） |
| 200102 | Sorftime API 返回空数据 | 检查关键词 |
| 200201 | 1688 API 认证失败 | 检查 app_key/secret |
| 200202 | 1688 商品不存在 | 尝试其他关键词 |
| 200301 | GLM API 调用失败 | 自动切换到备用模型 |
