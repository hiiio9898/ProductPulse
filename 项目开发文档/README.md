建议按编号顺序阅读：立项（1）→ 需求（2）→ 设计（3-6）→ 工程规范（7-8）→ 上线（9-10）。

## 技术栈概览

- **后端**：Python 3.10 + FastAPI
- **前端**：React 18 + Ant Design 5
- **数据**：PostgreSQL 15 + Redis 7
- **任务**：Celery + Beat
- **部署**：Docker + Docker Compose + Nginx
- **AI**：智谱 GLM-5.2（多模型备用）
