## 6.2 页面设计要点

### 首页概览（Dashboard）
- **顶部 KPI 卡片**（4个）：今日推荐数、预警数量、待匹配 SKU、综合评分 TOP1
- **中间推荐区**：本周 TOP 5 推荐产品卡片（带评分和推荐理由）
- **底部趋势图**：近 7 天品类热度趋势折线图

### 选品中心
- **顶部**：筛选条件栏（品类下拉、评分区间、匹配状态）
- **主体**：数据表格（展示所有产品及其核心指标）
- **每行操作**：查看详情、匹配 1688、标记关注
- **右上角**：导出数据按钮（CSV/Excel）

### 竞品监控
- **监控列表**：显示你手动标记关注的产品
- **价格变动**：红色（上涨）/ 绿色（下跌）箭头 + 百分比
- **历史曲线**：可点击查看价格趋势图（近 7/30 天）

### AI日报
- **展示 AI 生成的完整日报内容**（Markdown 渲染）
- **底部**：模型切换下拉菜单
- **操作**：重新生成按钮
- **历史**：左上角日期选择器（查看历史日报）

### 配置中心（Tab 切换）
- Tab1 - 选品阈值：所有阈值的输入框 + 保存按钮
- Tab2 - AI模型：下拉菜单切换主力/备用模型 + 测试连通性按钮
- Tab3 - SKU关联管理：手动关联/解除关联的操作列表
- Tab4 - 风险规则：规则列表 + 新增自定义规则表单

## 6.3 页面状态规范

每个列表/数据页面必须定义以下 5 种状态，禁止只画"有数据"态：

| 状态 | 表现 | 示例 |
|------|------|------|
| 加载中 | AntD `<Skeleton>` 骨架屏，禁止整页空白 | 选品表格骨架 |
| 空数据 | 插图 + 文案 + 引导操作按钮 | "暂无符合产品，试试调整阈值" |
| 错误 | 红色提示 + 重试按钮 | "数据加载失败，点击重试" |
| 数据延迟 | 顶部黄色 banner 提示 | "当前展示昨日数据（源接口延迟）" |
| 无权限 | 跳转登录/提示 | 单用户系统基本不会触发，保留兜底 |

### 交互细节
- 所有列表默认按综合评分降序，支持点击表头切换排序。
- 删除/解除关联等破坏性操作必须二次确认（AntD `Modal.confirm`）。
- 表单保存成功后用 AntD `message.success` 轻提示（1.5s），不弹模态框。
- 长任务（数据同步、日报生成）点击后按钮进入 loading + 禁用，完成后通知。

## 6.4 设计 Token

### 颜色系统

| 用途 | 色值 |
|------|------|
| 成功/利好 | #52C41A (绿色) |
| 警告/注意 | #FAAD14 (黄色) |
| 危险/风险 | #FF4D4F (红色) |
| 信息/提示 | #1890FF (蓝色) |
| 背景 | #F0F2F5 |
| 卡片背景 | #FFFFFF |
| 主文字 | #262626 |
| 辅助文字 | #8C8C8C |

### 字体系统

| 用途 | 字号 | 字重 | 行高 |
|------|------|------|------|
| 页面标题 H1 | 20px | 600 | 1.4 |
| 区块标题 H2 | 16px | 600 | 1.5 |
| 正文 | 14px | 400 | 1.5715 |
| 辅助文字 | 12px | 400 | 1.5 |

- 字体族：`-apple-system, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`
- 等宽（数字/代码）：`"JetBrains Mono", "Cascadia Code", monospace`

### 间距 / 圆角 / 阴影

| Token | 值 |
|-------|-----|
| 间距单位 | 4px 基准（8/12/16/24/32） |
| 圆角 | 卡片 8px / 按钮 6px / 输入框 4px |
| 阴影 | `0 2px 8px rgba(0,0,0,0.09)`（卡片悬浮） |
| 图标库 | `@ant-design/icons`（与 AntD 配套） |

## 6.5 组件规范（基于 AntD）

| 场景 | 组件 | 关键约定 |
|------|------|---------|
| 数据表格 | `Table` | 服务端分页；数字列右对齐；风险列用 `Tag` 染色 |
| 筛选条件 | `Form` + `Select`/`InputNumber` | 折叠收起；"重置"按钮清空所有条件 |
| 价格预警 | `Tag`/`Badge` | 上涨红、下跌绿，带箭头图标 |
| 趋势图 | `@ant-design/charts` 折线图 | 支持 7/30 天切换，tooltip 显示精确值 |
| AI 日报 | `react-markdown` 渲染 | 代码块高亮；emoji 正常显示 |
| 表单校验 | `Form` rules | 必填即时校验，提交前再次校验 |

## 6.6 可访问性（参考 WCAG 2.1 AA）

- 文字与背景对比度 ≥ 4.5:1（辅助文字 #8C8C8C 用于大字或非关键信息时需复核）。
- 所有可交互元素支持键盘导航（Tab/Enter/ESC），焦点态可见。
- 表单控件配 `aria-label`，图表补充 `alt` 文本摘要。
- 颜色不作为唯一信息载体（预警同时用图标 + 文字）。
